# Happy Scribbles


일상의 작은 행복들을 기록하는 일기장, **Happy Scribbles**

기록하기 쉽고 직관적인 인터페이스를 통해 매일의 기쁨과 중요한 순간들을 손쉽게 기록하고 관리할 수 있습니다.

## 소개

- 간편한 일기 작성 및 관리
- 체크리스트 추가 및 검색
